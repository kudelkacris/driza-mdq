🚀 SIGUIENTES PASOS:

1. Lee: INICIO_RAPIDO.md (está dentro de esta carpeta)
2. Abre la carpeta en VS Code
3. Terminal: npm install
4. Terminal: npm run dev
5. Abre: http://localhost:3000
6. ¡Prueba la tienda!
7. Cuando esté perfecto → Deploy a Vercel (GRATIS)
